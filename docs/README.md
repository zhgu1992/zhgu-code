# Docs 导航

[`docs/`](./) 是 rewrite 项目的唯一生效文档区（source of truth）。

## 目录约定

1. [`architecture/`](./architecture/): 当前已落地的架构事实与模块设计
2. [`adr/`](./adr/): 架构决策记录（ADR）
3. [`roadmap/`](./roadmap/): 规划、阶段目标、执行进度
4. [`learn/`](./learn/): 学习型文档与专题研究
5. [`learn/reader-guide/`](./learn/reader-guide/): 面向读者的阶段导读（讲核心功能、设计思路、问题-解法映射）
