# Docs 导航

`docs/` 是 rewrite 项目的唯一生效文档区（source of truth）。

## 目录约定

1. `architecture/`: 当前已落地的架构事实与模块设计
2. `adr/`: 架构决策记录（ADR）
3. `roadmap/`: 规划、阶段目标、执行进度
4. `learn/`: 学习型文档与专题研究

## 快速入口

1. 架构事实: `docs/architecture/system-design.md`
2. 总路线图: `docs/roadmap/master-roadmap.md`
3. 分阶段执行: `docs/roadmap/README.md`
4. 追踪模型: `docs/trace-model.md`
5. ADR 索引: `docs/adr/`
